Creating a Cell

To create a Cell, we first need to create a Lattice. There are multiple ways of doing it. For example, if we know the six lattice constants, we can do

julia> a = 4u"nm"ERROR: LoadError: UndefVarError: `@u_str` not defined
in expression starting at REPL[1]:1
julia> b = 180u"bohr"ERROR: LoadError: UndefVarError: `@u_str` not defined
in expression starting at REPL[2]:1
julia> c = 3u"angstrom"ERROR: LoadError: UndefVarError: `@u_str` not defined
in expression starting at REPL[3]:1
julia> lattice = Lattice(a, b, c, 90, 90, 90)ERROR: UndefVarError: `Lattice` not defined

Or, equivalently,

julia> Lattice([
           4u"nm" 0u"m" 0.0u"cm"
           0u"cm" 180.0u"bohr" 0u"m"
           0u"bohr" 0u"nm" (3//1)*u"angstrom"
       ])ERROR: LoadError: UndefVarError: `@u_str` not defined
in expression starting at REPL[1]:2

Then we can add atoms and their positions (in crystal coordinates):

julia> lattice = [
           -3.0179389205999998 -3.0179389205999998 0.0000000000000000
           -5.2272235447000002 5.2272235447000002 0.0000000000000000
           0.0000000000000000 0.0000000000000000 -9.7736219469000005
       ]3×3 Matrix{Float64}:
 -3.01794  -3.01794   0.0
 -5.22722   5.22722   0.0
  0.0       0.0      -9.77362
julia> positions = [[2 / 3, 1 / 3, 1 / 4], [1 / 3, 2 / 3, 3 / 4]]2-element Vector{Vector{Float64}}:
 [0.6666666666666666, 0.3333333333333333, 0.25]
 [0.3333333333333333, 0.6666666666666666, 0.75]
julia> atoms = [1, 1]2-element Vector{Int64}:
 1
 1
julia> cell = Cell(lattice, positions, atoms)ERROR: UndefVarError: `Cell` not defined

Reciprocal space

To get the reciprocal lattice, we run reciprocal:

julia> reciprocal(lattice)ERROR: UndefVarError: `reciprocal` not defined
Note

Never use ReciprocalLattice directly unless you know what you are doing!

Supercell generation

We can specify the replication factors in each direction in the following ways:

julia> supercell(lattice, [2, 3, 4])ERROR: UndefVarError: `supercell` not defined
julia> supercell(lattice, 3)ERROR: UndefVarError: `supercell` not defined
julia> supercell(cell, [2, 3, 4])ERROR: UndefVarError: `cell` not defined
julia> supercell(cell, 3)ERROR: UndefVarError: `supercell` not defined

If only one integer is provided, it will be used in all three spatial directions.

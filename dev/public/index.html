    CrystallographyBase.Cell ‚Äî Type
    Cell(lattice, positions, atoms)

    Create a new cell.

    Argument lattice is a Lattice type. Fractional atomic positions positions are given by a vector of $N$ vectors with floating point values, where $N$ is the number of atoms. Argument atoms is a list of $N$ values, where the same kind of atoms need to be the same type.

    CrystallographyBase.MonkhorstPackGrid ‚Äî Type
    MonkhorstPackGrid(mesh, is_shift)

    Represent the Monkhorst‚ÄìPack grid.

    Arguments

    • mesh: A length-three vector specifying the k-point grid ($nk_1 √ó nk_2 √ó nk_3$) as in Monkhorst‚ÄìPack grids.
    • is_shift: A length-three vector specifying whether the grid is displaced by half a grid step in the corresponding directions.

    CrystallographyBase.@m_str ‚Äî Macro
    m_str(s)

    Generate the Miller indices or Miller‚ÄìBravais indices quickly.

    Examples

    julia> m"[-1, 0, 1]"
3-element Miller:
 -1
  0
  1

julia> m"<2, -1, -1, 3>"
4-element MillerBravais:
  2
 -1
 -1
  3

julia> m"(-1, 0, 1)"
3-element ReciprocalMiller:
 -1
  0
  1

julia> m"(1, 0, -1, 0)"
4-element ReciprocalMillerBravais:
  1
  0
 -1
  0

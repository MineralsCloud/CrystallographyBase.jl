Creating a Cell

To create a Cell, we first need to create a Lattice. There are multiple ways of doing it. For example, if we know the six lattice constants, we can do

julia> using CrystallographyBase, Unitful, UnitfulAtomic
julia> lattice‚ÇÅ = Lattice(4u"nm", 180u"bohr", 3u"angstrom", 90, 90, 90)Lattice{Unitful.Quantity{Float64, ùêã, Unitful.FreeUnits{(m,), ùêã, nothing}}}
 4.0e-9 m  0.0 m  0.0 m
 0.0 m  9.525189796254e-9 m  0.0 m
 0.0 m  0.0 m  3.0e-10 m

Or, equivalently,

julia> lattice‚ÇÅ == Lattice([
           4u"nm" 0u"m" 0.0u"cm"
           0u"cm" 180.0u"bohr" 0u"m"
           0u"bohr" 0u"nm" (3//1)*u"angstrom"
       ])true

Then we can add atoms and their positions (in crystal coordinates):

julia> lattice‚ÇÇ = [
           -3.0179389205999998 -3.0179389205999998 0.0000000000000000
           -5.2272235447000002 5.2272235447000002 0.0000000000000000
           0.0000000000000000 0.0000000000000000 -9.7736219469000005
       ]3√ó3 Matrix{Float64}:
 -3.01794  -3.01794   0.0
 -5.22722   5.22722   0.0
  0.0       0.0      -9.77362
julia> positions = [[2 / 3, 1 / 3, 1 / 4], [1 / 3, 2 / 3, 3 / 4]]2-element Vector{Vector{Float64}}:
 [0.6666666666666666, 0.3333333333333333, 0.25]

Reciprocal space

To get the reciprocal lattice, we run reciprocal:

julia> reciprocal(lattice‚ÇÅ)ReciprocalLattice{Unitful.Quantity{Float64, ùêã^-1, Unitful.FreeUnits{(m^-1,), ùêã^-1, nothing}}}(Unitful.Quantity{Float64, ùêã^-1, Unitful.FreeUnits{(m^-1,), ùêã^-1, nothing}}[2.4999999999999997e8 m^-1 0.0 m^-1 0.0 m^-1; 0.0 m^-1 1.0498478470143168e8 m^-1 0.0 m^-1; 0.0 m^-1 0.0 m^-1 3.333333333333333e9 m^-1])
Note

Never use ReciprocalLattice directly unless you know what you are doing!
